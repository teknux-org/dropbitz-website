How does it work ?

There's nothing to install, every Web browser is compatible with DropBitz


  • Start

    You generate a secure code

    A link containing the code is sent to your customer


  • Notify

    Your customer receives an email

    He clicks on the link to open the web browser, he can start sharing files with you


  • Share

    Customer now drags & drop a file on the page

    The file is uploaded and you receive a notification when it's done


  • Done

    The file is now available for you

    No need to download it again, it is available on your infrastructure

Benefits

Secure

Generate upload/download codes for single use, or for a specific amount of time. All transfers are encrypted.

Big files

Reliably transfer files of hundreds of gigabytes. Transferred files are stored on your network and available immediately

Easy to use

Nothing to install for your customer! Just drag the file in a Web browser. Operators can generate a code in seconds.

Flexible

DropBitz is a breeze to install and ready to use in minutes. DropBitz supports active Directory and LDAP.
