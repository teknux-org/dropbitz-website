DropBitz

Share files securely with your customers directly on your network !

Find Out More

DropBitz allow your customer to securely upload files in one drag&drop!

How does it work ?

  1. You generate a secure code
  2. A link containing the code is sent to your customer
  3. Your customer clicks on the link to open his web browser
  4. The customer now drags & drop a file on the Web page
  5. The file is uploaded and you receive a notification when it's done
  6. The file is available on your network (no need to download it again)

There's nothing to install, every Web browser is compatible with DropBitz

Benefits

Secure

Generate upload/download codes for single use, or for a specific amount of time. All transfers are encrypted.

Big files

Reliably transfer files of hundreds of gigabytes. Transferred files are stored on your network and available immediately

Easy to use

Nothing to install for your customer! Just drag the file in a Web browser. Operators can generate a code in seconds.

Flexible

DropBitz is a breeze to install and ready to use in minutes. DropBitz supports active Directory and LDAP.
